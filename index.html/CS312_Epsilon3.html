The saga's epic conclusion is out NOW!

Epsilon 3 is a third-person 3D RPG Action-Adventure game where the player can create their own character and go on adventures, either solo or online with friends!

The game has three options available for purchase: Regular Version (Base Game), Regular Version with added DLC, and the Deluxe Version (see "Pricing" page for current prices of each version).
